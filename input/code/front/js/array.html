Array.from() 是 JavaScript 中的一个静态方法，用于从类似数组或可迭代的对象创建一个新的数组实例。

将类似数组的对象转换为数组：有些对象看起来像数组（即有长度属性和索引元素），但并不是真正的数组。例如，arguments 对象或 DOM 集合（如 document.querySelectorAll() 返回的对象）。Array.from() 可以将这些对象转换为真正的数组。

将可迭代对象转换为数组：Array.from() 可以从任何可迭代对象（如 Set、Map 或其他实现了迭代器协议的对象）创建一个数组。

映射类似数组的对象到新的数组：你可以提供一个映射函数作为 Array.from() 的第二个参数，以在创建新数组时对每个元素进行转换。

基本用法：

js
const arrayLike = { length: 3, 0: "a", 1: "b", 2: "c" };
const arr = Array.from(arrayLike);
console.log(arr); // 输出: ['a', 'b', 'c']

const numbers = { length: 3, 0: 1, 1: 4, 2: 9 };

参数：

  • arrayLike (必需):

    类型：类数组对象或可迭代对象。

    描述：你想要转换为数组的对象。这可以是一个类似数组的对象（有 length 属性和按索引访问的元素），或者是任何可迭代的对象（例如 Set、Map、字符串、数组等）。

  • mapFn (可选):

    类型：函数。

    描述：如果提供了此参数，Array.from() 会使用该函数处理 arrayLike 中的每个元素，并返回由这些处理后的元素组成的新数组。这个函数的参数与 Array.prototype.map() 中的回调函数相同。

  • thisArg (可选):

    类型：任何值。

    描述：如果提供了 mapFn，则 thisArg 会作为回调函数的 this 值。如果未提供 thisArg，则 undefined 会被用作 this 值。

示例：

js
const li = document.querySelectorAll("li");
const list = [];
const newList = Array.from(
  li,
  function (item, index) {
    // 不用箭头函数，因为箭头函数没有this，要改变this，得用普通函数
    console.log(this); // this是list这个数组
    // item是<li>a</li>这些dom元素
    return index;
  },
  list
);
// newList是[1,2,3]，因为这里返回的是下标

of ​

Array.of() 是 JavaScript 中的一个静态方法，用于创建一个新的数组实例，无论参数的数量或类型如何。这个方法总是返回一个新的数组，而不是一个类似数组的对象。

普通用法：

js
// 普通数组
const arr = Array.of(1, 2);
// [1,2]
// 空数组
const empty = Array.of();
// []

与 new Array 的区别 ​

尽管 new Array() 也可以创建一个新数组，但它对单一参数的处理方式与 Array.of() 不同：

js
const arr1 = new Array(1); // 创建一个包含单个元素 1 的数组
const arr2 = Array.of(1); // 创建一个包含单个元素 1 的数组

const arr3 = new Array(1, 2, 3); // 创建一个包含元素 1, 2, 3 的数组
const arr4 = Array.of(1, 2, 3); // 创建一个包含元素 1, 2, 3 的数组

const arr5 = new Array("foo"); // 创建一个包含一个字符串 'foo' 的数组
const arr6 = Array.of("foo"); // 创建一个包含一个字符串 'foo' 的数组

// 使用单个数字参数时，new Array() 和 Array.of() 的行为不同
const arr7 = new Array(3); // 创建一个长度为 3 的空数组
const arr8 = Array.of(3); // 创建一个包含单个数字 3 的数组

find ​

arr.find，查找数组中满足要求的第一个元素的值。

参数：

  • callback：

    element:当前遍历的元素。

    index：可选，当前遍历到的索引。

    array：可选，数组本身。

  • thisArg，可选：

    执行回调时用作 this 的对象。

返回值：数组中第一个满足所提供测试函数的元素的值，否则返回 undefined。

示例：

js
const li = [1, 2, 3, 4, 5];
const val = li.find((element) => {
  if (element > 2) {
    return true;
  }
});
// val是3
// 或者简写：
const r = li.find((ele) => ele >= 2);
