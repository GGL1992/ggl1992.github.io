注释

n

名词（Noun,简称 n.），是词类的一种，属于实词，名词表示人、事物、地点或抽象概念的名称，名词同时也分为专有名词和普通名词。

v

动词（Verb）一般就是用来表示动作或状态的词汇。在英语中，动词按作用和功能主要分为两大类，一类是谓语动词，另一类是非谓语动词。

adj

形容词（Adjective），很多语言中均有的主要词类中的一种。形容词主要用来描写或修饰名词或代词，表示人或事物的性质、 状态、特征或属性，常用作定语，也可作表语、补语或状语。

adv

副词（Adverb 简称 adv.）是指在句子中表示行为或状态特征的词，用以修饰动词、形容词、其他副词或全句，表示时间、地点、程度、方式等概念。副词可分为：时间副词、频率副词、地点副词、方式副词、程度副词、疑问副词、连接副词、关系副词、表顺序、表完成的副词。

pron

代词是代替名词或一句话的一种词类。大多数代词具有名词和形容词的功能。英语中的代词，按其意义、特征及在句中的作用分为：人称代词、物主代词、指示代词、反身代词、相互代词、疑问代词、关系代词、连接代词、不定代词和替代词十种。

conj

连词（conj conjunction）是一种虚词， 它不能独立担任句子成分而只起连接词与词，短语与短语以及句与句的作用。连词主要可分为 4 类：并列连词.转折连词.选择连词和因果连词。连词也可以分为 2 类：并列连词和从属连词。

“Conjunction”是一个连词，用于连接两个或多个单词、短语或从句，以表示它们之间的关系。例如，“and”是一个连词，用于连接两个并列的单词或短语；“because”是一个连词，用于表示原因或理由。

abbr

“abbr.”是“abbreviation”的缩写。

“Abbreviation”是指一个或多个单词的简写形式，通常用于表示一个完整的概念或词组。例如，“Mr.”是一个缩写，表示“Mister”；“Jan.”是一个缩写，表示“January”。

因此，“abbr.”通常用于表示一个缩写或简写形式。

symb

“symb.”是“symbol”的缩写。

“Symbol”是指一个代表或表示另一个事物的标记或符号。它可以是一个字母、数字、图形或其他任何东西，用于表示一个概念、事物或属性。例如，“#”是一个符号，用于表示数字“16”。

因此，“symb.”通常用于表示一个符号或标记。

int

“int.”是“interjection”的缩写。

“Interjection”是一个感叹词或表达强烈情感的词，通常用于表达惊讶、喜悦、悲伤等情感。例如，“Oh!”是一个感叹词，表示惊讶或失望；“Wow!”是一个感叹词，表示惊喜或兴奋。

因此，“int.”通常用于表示一个感叹词或表达情感的词。

det

“det.”是“determiner”的缩写。

“Determiner”是一个限定词，用于限定名词或名词短语，以表示其范围或特定性。例如，“the”是一个限定词，用于表示特定的一个或一组事物；“some”是一个限定词，用于表示一些或某些事物。

因此，“det.”通常用于表示一个限定词或限定短语。

vt

表示及物动词（transitive verb）。及物动词是指后面需要跟宾语来构成完整的谓语的动词。例如：write（写）、eat（吃）、play（玩）、read（读）。
