scanner ​

如何从键盘获取不同类型（基本数据类型、string 类型）的变量？使用 scanner 类。

使用 scanner 获取不同类型数据的步骤：

  • 导包：import java.util.Scanner
  • 提供或创建一个 Scanner 类的实例
  • 调用 Scanner 类中的方法，获取指定类型的变量(方法都是 nextXXX()的格式，字符串的是 next())，这是阻塞式方法，一旦执行，则等待输入，不输入，下面的语句就不会执行。
  • 释放资源，调用 Scanner 类的 close()，防止内存泄漏

注意：需要根据相应的方法，来输入指定类型的值，如果输入的数据类型与要求的类型不匹配时，会报异常导致程序终止。

Scanner 类中提供了获取 byte、short、int、long、float、double、boolean、string 类型变量的方法。

注意：没有提供获取 char 类型变量的方法，需要使用 next().charAt(0)来获取。

案例：小明注册某交友网站，要求录入个人相关信息，如下：

请输入你的网名，你的年龄，你的体重，你是否单身，你的性别等情况。

java
import java.util.Scanner;

class Test{
    public static void main(String[] args) {
        // 创建实例
        Scanner scan = new Scanner(System.in);
        // 调用Scanner类中的方法，获取指定类型的变量
        System.out.println("欢迎光临");
        System.out.print("请输入网名:");
        String name = scan.next();
        System.out.print("请输入年龄：");
        int age = scan.nextInt();
        System.out.print("请输入体重：");
        double weight = scan.nextDouble();
        System.out.print("请输入是否单身（单身：true，不单身：false）");
        boolean isSingle = scan.nextBoolean();
        System.out.print("请输入你的性别：(男/女)");
        char gender = scan.next().charAt(0);
        // 判断字符串是否相等
        if (name.equals("高国良")) {
            System.out.println("666");
        }
        // 关闭资源，调用Scanner类的close()，防止内存泄漏
        scan.close();
    }
}

字符串的相等判断 ​

字符串的相等判断要用equals方法，而不能用==，因为等号看的是内存地址是否相等，equals 方法看的是内容是否相等。

字符串是引用类型，比如有String name = "高"，如果用name == "高"，右边的值是一个字面常量，它和 name 的地址不一样，所以不相等。

获取随机数 ​

如何获取一个随机数？

可以使用 java 提供的 API：Math 类的 random 方法。

random()调用以后，会返回一个double类型的随机数，范围是[0.0,1.0)，开闭区间。

如何获取一个[a,b]区间的随机整数？

公式：(int)(Math.random() * (b - a + 1)) + a。

如何获取一个[a,b)区间的随机整数？

公式：(int)(Math.random() * (b - a)) + a。

题目：获取一个[0,100]范围的随机整数

(int)(Math.random() * 101)

题目：获取一个[1,100]范围的随机整数

(int)(Math.random() * 100) + 1

switch-case ​

语法格式:

switch (表达式) {
    case 常量1：
        // 执行语句
        // 可选的break
    case 常量2：
        // 执行语句
        // 可选的break
    default:
        // 执行语句
        // 可选的break
}

执行过程：

根据表达式中的值，一次匹配 case 语句，一旦与某一个 case 中的常量相等，那么就执行此 case 中的执行语句。

执行完执行语句之后：

  • 遇到 break，则执行 break 后，跳出当前 switch-case 结构
  • 没有遇到 break，则继续执行其后的 case 中的执行语句（case 穿透），直到遇到 break 或者执行完所有的 case 及 default 中的语句，退出当前的 switch-case 结构。

说明：

  • switch 中的表达式只能是特定的数据类型。如下：byte、short、char、int、枚举（JDK5.0 新增）、string（JDK7.0 新增）
  • case 后都是跟的常量，使用表达式与这些常量做相等的判断，不能进行范围的判断。
  • 开发中，使用 switch-case 时，通常 case 匹配的情况都有限。
  • break：可以使用在 Switch-case 中，一旦执行此 break 关键字，就跳出当前的 switch-case 结构
  • default：类似于 if-else 中的 else 结构。default 是可选的，可以没有。而且位置是灵活的，无限制，可以放在任何位置，甚至可以放在最前面，建议放在最后面。

示例：

java
int num = 1;
// 打印1
switch (num) {
    default:
        System.out.println("default");
        break;
    case 1:
        System.out.println("1");
        break;
    case 2:
        System.out.println("2");
        break;
}

if 与 switch 细节比较 ​

switch 和 if-else 都可用的情况下，建议使用 switch，因为 switch 效率稍高。if-else 因为可能有范围的匹配，它的机制时间复杂度是O(n)，而 switch 只是做简单的相等比较，它的时间复杂度是常数级O(1)。

细节比较：

if-else 语句优势：

  • if 语句的条件是一个布尔类型值，if 表达式为 true 进入分支，可以用于范围的判断，也可以用于等值的判断，使用范围要广。
  • switch 语句的条件是一个常量值（byte/short/int/char/枚举/string），只能判断某个变量或表达式的结果是否等于某个常量值，使用场景狭窄。

switch 语句优势：

  • 当条件是判断某个变量或表达式是否等于某个固定的常量值时，使用 if 和 switch 都可以，习惯上使用 switch 更多，因为效率稍高。当条件是区间范围的判断时，只能使用 if 语句。
  • 使用 switch 可以利用穿透性，同时执行多个分支（case 穿透）,而 if-else 没有穿透性。

循环语句 ​

理解：循环语句具有在某些条件满足的情况下，反复执行特定代码的功能。

循环结构分类：

  • for 循环
  • while 循环
  • do-while 循环

循环结构四要素：

  • ① 初始化部分
  • ② 循环条件部分
  • ③ 循环体部分
  • ④ 迭代部分

从循环次数角度分析：

  • do-while 循环至少执行一次循环体语句。
  • for 和 while 循环先判断循环条件语句是否成立，然后决定是否执行循环体。

如何选择：

  • 遍历有明显的循环次数（范围）的需求，选择 for 循环。
  • 遍历没有明显的循环次数（范围）的需求，选择 while 循环。
  • 如果循环语句块至少执行一次，可以考虑使用 do-while 循环。
  • 本质上，三种循环之间完全可以相互转换，都能实现循环的功能。

无限循环结构的使用：

while(true) {}
for(;;) {}
for(;true;) {}

for 循环和 while 循环可以相互转换！

for 循环和 while 循环的小区别：初始化条件的作用域范围不同，while 循环中的初始化条件在 while 循环结束后，依然有效。

练习：世界最高山峰是珠穆朗玛峰，它的高度是 8848.86 米，假如有一张足够大的纸，它的厚度是 0.1 毫米。请问，我折叠多少次，可以折成珠穆朗玛峰的高度？

java
// 声明珠峰和纸的厚度
double paper = 0.1; // 单位，毫米
double zf = 8848860; // 单位，毫米
// 定义一个变量，记录折纸的次数
int count = 0;
// 通过循环结构，不断调整纸的厚度（当纸的厚度超过珠峰高度时，停止循环）
while (paper <= zf) {
    paper *= 2;
    count++;
}
System.out.println("纸的高度为：" + (paper / 1000) + "，超过了珠峰的高度：" + (zf / 1000));
System.out.println("折纸次数：" + count);

嵌套循环 ​

所谓嵌套循环，是指一个循环结构 A 的循环体是另一个循环结构 B。比如 for 循环里面还有一个 for 循环，就是嵌套循环。其中，for/while/do-while 均可以作为外层循环或内层循环。

实质上，嵌套循环就是把内层循环当做外层循环的循环体。只有当内层循环的循环条件为 false 时，才会完全跳出内层循环，才可以结束外层的当次循环，开始下一次的外层循环。

设外层循环次数为 m 次，内层为 n 次，则内层循环体实际上需要执行 mxn 次。

技巧：从二维图形的角度看，外层循环控制行数，内层循环控制列数。

开发经验：实际开发中，我们最多见到的嵌套循环是两层，一般不会出现超过三层的嵌套循环。如果将要出现，一定要停下来重新梳理业务逻辑，重新思考算法的实现，控制在三层以内。否则可读性会很差。

练习：找出 100000 以内的所有质数（公约数只有 1 和自身的数）。

超高性能的实现，算法的魅力，比简单的循环查找，提高了一千多倍的效率，花费时间由几千毫秒变成了毫秒。

java
// 获取系统当前的时间
long start = System.currentTimeMillis();
boolean isFlag = true;
int count = 0; // 记录质数的个数
for (int i = 2; i <= 100000; i++) {
    // 遍历100000以内所有的自然数
    // 判定是否是质数
    for (int j = 2; j <= Math.sqrt(i); j++) {
        if (i % j == 0) {
            isFlag = false;
            break; // 对于非质数有效果
        }
    }
    if (isFlag) {
        count++;
    }
    // 重置isFlag
    isFlag = true;
}
// 获取系统当前的时间：
long end = System.currentTimeMillis();
System.out.println("质数的总个数为：" + count);
System.out.println("花费时间为：" + (end - start));

这个算法的关键是：Math.sqrt(i)，为什么呢？

一个数，假设 100：

它如果被 2 除，结果是 50，被 3 除，结果是 33.3...，被 4 除，结果是 25。

它如果被 50 除，结果是 2，被 25 除，结果是 4。

那么这个中心点是哪里呢？就是除以一个数，结果是这个数本身那个位置。将 100 开方，得到 10。

只需要遍历 10 左边的数就可以了，因为左右是对称的，除右边的数，不过是又得到了左边的数。

这样，计算量就极大的降低了。
